--- --Instruction of downloadable attendance checker-- ---
--- Manual Upload of Database: ---

STEP1: Go to downloadable "AttendanceSystem" file.

STEP2: Extract the AttendanceSystem File.

STEP3: Then copy "attendanceDB.mdf & attendanceDB_log.LDF" file.

STEP4: Paste it on the Local Disk (C:) > Program Files > Microsoft SQL Server > MSSQL10.SQLEXPRESS > MSSQL > DATA

STEP5: Then go to SQL Server Database and connect it.

STEP6: Check the Databases, if it's already attach the file.


--- Other way (2nd Instruction): Note: If the first instruction of the manual upload of Database didn't work, go to the 2nd Instruction. ---

STEP1: Create a file out side the Local Disk (C:)

STEP2: Then go to downloadable "AttendanceSystem" file.

STEP3: Extract the AttendanceSystem file.

STEP4: Click AttendanceSystem > AttendanceSystem > SQL Server Database.

STEP5: Then copy "attendanceDB.mdf & attendanceDB_log.LDF" file.

STEP6: Paste it on the Local Disk (C:) > File you create

STEP7: Properties of the both file > Security > Click Authenticated Users or Create Everyone > Click all Allow > Click the Read Only to edit the database.

STEP8: Then go to SQL Server Databases and connect it.

STEP9: Proceed to the Databases > Right Click > Attach the file.

STEP10: After that check the Databases , if it's already attach the file.


--- Change data source to run the system and access database: ---

STEP1: Open downloaded AttendanceSystem solution file on Visual Studio 2017.

STEP2: Double click every form and change the all Data Source into (YourDevice Name)\SQLEXPRESS;Initial Catalog=attendanceDB;Integrated Security=True;Pooling=False

STEP3: And paste it on every sql connection code block in each form.

STEP4: Then run the system to check if you change the data source successfully.


If you want to Publish the system for the easy acces, here the following below :

STEP1: Click the VB AttendanceSystem.

STEP2: Right Click > Properties > Publish.

STEP3: Click the Publishing Folder Location, you can choose where to put the system.

STEP4: Choose a Install Mode and Settings.

STEP5: Publish Version make it ( 1, 0, 0, 2).

STEP6: Click Publish Now.

STEP7: Wait the published system and try it.
